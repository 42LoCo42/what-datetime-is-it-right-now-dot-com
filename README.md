## [WhatDatetimeIsItRightNow.com](https://datetime.eleonora.gay)
A site that tells you what date and time it is right now -
resurrected, modernized and built with Nix!

### Run Dev
```shell
nix develop # or `direnv allow`
pnpm install
pnpm gatsby develop
```
